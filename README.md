![app_logo](./assets/no_image.png)

팝업 정보를 한 눈에!  
근처에서 열리는 팝업 스토어 정보를 손쉽게 찾아보세요.
# 🚀 Features
위치 기반 팝업 스토어 검색  
다양한 언어 지원 (한국어, 영어, 중국어)  
광고를 통한 지속 가능한 서비스 운영  
심플하고 직관적인 UI  

# 🛠️ Tech Stack
Flutter  
Google Maps API  
AdMob (광고 수익화)  
dotenv (환경변수 관리)

- 🇺🇸 English
